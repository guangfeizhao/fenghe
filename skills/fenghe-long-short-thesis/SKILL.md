# 风和多空投资论证框架

## 这个框架做什么？

> "只要你深入企业分析，就是投资；只要你猜市场方向，就是赌博。"

风和是多空策略对冲基金。每一个投资标的都需要同时论证做多和做空两面——即使最终只选择一个方向，充分理解对手盘的逻辑是投资成功的关键。本框架帮助分析师将5M分析和3C3D判断的结果，组织成结构化的投资论据和报告。

## 做多论证

### 类型一：结构性做多（做多鹰）

公司正在赢取市场份额、扩张利润率，管理团队卓越。这是最"正统"的做多，驱动力来自D2（外延变化）中的结构性因素。

**论证要点**：
1. 鹰的特征（来自5M中的M2/M3/M5）：份额趋势、利润率趋势、团队纪录
2. 确定性评估（C3）：关键假设是否可验证
3. 催化剂（T1/T2）：近期和中期的驱动事件
4. 估值安全边际（D3）：当前估值水平和下行保护
5. 缺陷及应对：已识别的缺陷、量化恶化后果、退出触发条件
6. 仓位建议：权重、止损线、预期回报、持有周期

### 类型二：逆势做多（周期底部）

> "华尔街总会因为一两个季度没有达到贪婪的分析师们的预期就低价抛售一些极有远景的企业。"

核心逻辑是"逆短期基本面之势，顺长期基本面之势"。这种做多需要很强的判断力——你要确认这是周期性下跌而非结构性下跌。

**论证要点**：
1. 确认是周期性下跌而非结构性下跌（这是最关键的判断）
2. 确认长期ROE可持续
3. 短期利空已充分反映在股价中
4. 市场共识过度悲观（D3机会）
5. 有T1/T2内恢复的催化剂

### 类型三：缺陷投资

> "完美的企业鲜有，偶有完美的企业，价钱通常是高不可攀的。"

投资有缺陷但可跟踪、可监控、有修复可能的企业。这是风和最常用的做多方式——因为现实中几乎所有投资都是缺陷投资。

**缺陷投资六要素检查**：
- 看见了缺陷并能评估其恶化后果
- 缺陷可跟踪、可监控
- 缺陷"与时间为友"或可被团队修复
- 不是"火中取栗"（缺陷不在恶化中）
- 管理团队有"过五关斩六将"的纪录
- 组合分散，非集中持有

## 做空论证

> "我愿意做空曾经伟大的企业。"

### 类型一：宴席理论（Party Over）

> "天下没有不散的宴席。只要哪里有疯狂的party，哪里就有泡沫，哪里就有做空的机会。"

行业或主题的泡沫即将破灭。论证要点：行业狂欢的证据（估值泡沫、产能疯狂扩张、全民参与）、散场信号（需求见顶、政策转向、利率上行）、风口消散后猪的脆弱性。

### 类型二：踩烂理论（Stepping on the Broken）

> "有些已经变烂的企业，还会变得更烂。"

基本面已开始恶化，市场只反映了第一波，后续还有更多恶化。这是一种"趋势做空"——关键洞察是"看似便宜"往往是陷阱。

**踩烂的典型路径**：
```
估值下跌（先知先觉卖出）
    ↓
估值看似便宜 → 抄底者/空头平仓 → 反弹
    ↓
盈利恶化逐季确认 → 股价下跌但估值反而上升
    ↓
继续恶化直至亏损 → 这就是"踩烂"
```

**条件**：对基本面继续恶化有确定性判断、理解做空数学、不因静态估值便宜而平仓。

### 类型三：结构性做空（M2做空）

市场份额正在被蚕食，商业模式被颠覆。论证要点：需要份额下滑的**数据证据**（不仅是逻辑推演）、背后原因分析、管理层反击可能性评估、被并购价值考量。

### 类型四：周期性做空（M3做空）

利润率创新高但供需即将反转。这种做空确定性较高（因为管理层也无能为力），但要注意这些公司过几年又是好汉一条——踩烂需谨慎。

### 做空通用注意事项

> "做空和做多的主要区别是，做空更需要耐心等待进入时机。"

- 纯估值做空（没有基本面恶化配合）必须有T1/T2催化剂
- 时间常常是做多的朋友，但不一定是做空的朋友
- 做空的对手不只是其他交易者，还有企业家（精英分子）——他们有强烈的动机和能力反击
- 成功做空的平均回报有限，靠累积小胜
