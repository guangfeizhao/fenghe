# 风和5M价值分析框架

## 为什么用5M？

风和基金创始人胡猛认为，**公司的核心价值就是ROE——为股东创造利润的能力**。但ROE是一个结果指标，直接看ROE数字很难判断它是否可持续、是否会变化。5M框架把ROE拆解为5个可独立跟踪的维度，让分析师能看清ROE背后的驱动力，并据此做出投资判断。

> "价值就是企业的现金 + 可变现的资产 + 创造未来现金的能力（硬核ROE）。"

## 5M各维度分析指引

### M1 - 目标市场（Target Addressable Market）

**要回答的核心问题**：这家公司参加的是什么party？party还能开多久？

这是最宏观的一层。一个市场处于高速扩张期时，几乎所有参与者都能增长（"鹰猪齐飞"）；但当市场见顶，没有护城河的企业（"猪"）会率先坠落。分析M1的目的是判断公司所在赛道的天花板和生命阶段。

分析时关注以下方面：

- **TAM量化**：用"自上而下"和"自下而上"两种方法交叉验证市场规模。两种方法差距过大时，说明某个假设可能有问题。
- **渗透率**：当前渗透率与成熟市场的对标（例如中国人均某类消费 vs 日韩水平），渗透率低意味着空间大。
- **增速趋势**：市场增速是在加速、减速还是已经转负？增速的二阶导比绝对增速更重要——增速开始放缓往往是拐点信号。
- **风口判断**：区分"风口型市场"（增长靠M1驱动，潮水退去猪会现形）和"结构性市场"（增长靠企业自身M2/M3能力驱动）。

**多空信号**：
- 做多信号：TAM仍在扩张，渗透率远低于成熟市场
- 做空信号：M1见顶或收缩，风口上的猪即将坠落（"天下没有不散的宴席"）

### M2 - 市场份额（Market Share）

**要回答的核心问题**：这是鹰还是猪？能不能独霸一片天空？

M2是区分"鹰"和"猪"的关键维度。在M1驱动的市场里，猪也能飞；但只有鹰在风停之后还能飞。判断一家公司是鹰还是猪，最有说服力的证据就是市场份额的变化趋势。

分析时关注以下方面：

- **份额趋势**：过去8-12个季度的市场份额变化曲线。月度或季度频率的跟踪数据最有价值，因为份额趋势的持续性比单点数据重要得多。
- **集中度**：行业CR3/CR5，是否已形成寡头格局。寡头格局中的头部企业是天然的做多候选。
- **份额获取方式**：靠价格战抢来的份额不可持续，靠产品品质和创新赢来的份额才是结构性优势。
- **逆境中的表现**：下行周期中仍在扩张份额的企业，几乎可以确认是"鹰"——这是最强的做多信号之一。

**多空信号**：
- 做多信号：份额持续上升，特别是在行业下行期仍在扩张
- 做空信号：份额被蚕食，特别是被后发优势的新进入者抢夺（结构性做空的基础）

### M3 - 利润率（Operating Profit Margin）

**要回答的核心问题**：利润率处于周期的什么位置？方向朝哪？

利润率是周期性最强的维度。高利润率吸引竞争者进入，最终把利润率压低；反之，低利润率逼退竞争者，为幸存者创造扩张空间。理解利润率的周期位置，是判断"做多"还是"做空"的关键。

分析时关注以下方面：

- **利润率趋势**：毛利率、营业利润率、净利率的8-12季度趋势。趋势比绝对水平更重要。
- **驱动因素**：是定价权驱动、成本结构改善、规模效应，还是产品组合优化？不同驱动因素的可持续性差异很大。
- **周期位置**：当前利润率在历史区间中的什么位置？接近历史高位时要警惕均值回归，接近历史低位时要寻找反转信号。
- **同行对比**：利润率差距是在扩大还是缩小？扩大说明竞争优势在增强。

**多空信号**：
- 做多信号：利润率从底部回升，或因结构性优势持续扩张
- 做空信号（周期性）：利润率创新高但供需即将反转——逆势做空利润率高位

### M4 - 商业模式（Business Model）

**要回答的核心问题**：这个生意的本质是什么？护城河有多深？

M4评估的是企业盈利能力的"基因"。好的商业模式能让ROE持续维持在高位，差的商业模式即使一时盈利也难以持续。

分析时关注以下方面：

- **收入模式**：一次性收入 vs 经常性收入（订阅/SaaS），ToB vs ToC vs ToG。经常性收入更可预测，确定性更高。
- **护城河来源**：品牌、规模效应、网络效应、转换成本、专利/牌照。护城河越深，ROE的可持续性越强。
- **资本效率**：轻资产 vs 重资产，ROIC水平。轻资产模式通常有更高的ROE和更好的抗周期能力。
- **颠覆风险**：商业模式是否正在被新技术或新模式颠覆？
- **先发 vs 后发**：科技产品常常是后发优势——这是风和的一个重要发现，分析科技股时需要特别留意。

**多空信号**：
- 做多信号：强护城河 + 轻资产 + 高ROIC
- 做空信号：商业模式被颠覆，或过度依赖已消失的风口

### M5 - 管理团队（Management Team）

**要回答的核心问题**：这个团队能不能"过五关斩六将"？

M5是最主观但也可能最关键的维度。胡猛强调，管理层的"主观能动力"是做多时间的朋友、做空的对手。评估管理层不是看简历，而是看他们在关键时刻的决策质量。

分析时关注以下方面：

- **战略能力**：历史上重大战略决策的质量——选对了几次，错过了几次？
- **执行能力**：说到做到的程度。承诺的目标有多少兑现了？
- **资本配置**：分红、回购、并购、再投资的历史纪录。资本配置差的管理层会系统性地摧毁股东价值。
- **治理结构**：股权结构是否合理、独立性如何、合规记录。
- **X Factor**：管理层的韧性和斗志。这一点在做空分析中尤其重要——优秀的管理层可能力挽狂澜，导致做空失败。

**多空信号**：
- 做多信号：团队"过五关斩六将"的纪录优异——这是做缺陷投资的前提
- 做空信号：但做空时要充分评估管理层反击的可能性和被并购的价值

## 分析产出

对每个M维度进行1-10分评分，并给出趋势方向（↑ ↗ → ↘ ↓），产出一张评分卡：

```
┌─────────────────────────────────────────────────┐
│           5M价值分析评分卡                        │
├──────────┬───────┬────────┬─────────────────────┤
│ 维度     │ 评分  │ 趋势   │ 关键发现             │
├──────────┼───────┼────────┼─────────────────────┤
│ M1 市场  │  /10  │  ↑↗→↘↓ │                     │
│ M2 份额  │  /10  │  ↑↗→↘↓ │                     │
│ M3 利润率│  /10  │  ↑↗→↘↓ │                     │
│ M4 模式  │  /10  │  ↑↗→↘↓ │                     │
│ M5 团队  │  /10  │  ↑↗→↘↓ │                     │
├──────────┼───────┼────────┼─────────────────────┤
│ 综合     │  /10  │        │                     │
└──────────┴───────┴────────┴─────────────────────┘

ROE分析：当前ROE____%, 趋势____
估值：当前PE____x, PB____x, 历史分位____%
```

在评分卡之后，用1-2段文字概述最重要的发现和初步的多空倾向，为后续使用 3C3D 框架做时机判断或使用 LOGOS 做风险排查提供基础。

## 不同市场环境下的分析侧重

不同的市场环境下，5M各维度的重要性不同：

| 市场环境 | 做多侧重 | 做空侧重 |
|---------|---------|---------|
| 风口期（鹰猪齐飞）| M1, M4 | 暂不急于做空 |
| 结构性分化期 | M2, M3, M5 | M1见顶, M4被颠覆 |
| 下行周期 | M2逆势扩张的鹰 | M3周期见顶的品种 |

## 数据获取指引

优先使用可用的MCP数据连接器获取数据：
- **S&P Global / FactSet**：财务数据、市场份额数据、同行对比
- **Morningstar**：估值数据、护城河评级
- **新闻/电话会议**：管理层最新发言和策略信号

当无MCP连接器可用时，使用 web search 获取公司年报、季报、行业报告、管理层公开发言等公开信息。

## 分析纪律

- **永远先看ROE**：ROE是起点也是终点，5M是拆解ROE的工具，不要在5M分析中忘了最终要回答的是"ROE能否持续提升"。
- **关注变化而非状态**：一个M2=70%份额的公司不一定比M2=30%的公司更值得买——关键是份额在增长还是在下降。
- **不追求完美**：世上鲜有完美企业。缺陷投资是常态，但前提是看清缺陷、量化缺陷恶化的后果、并确认缺陷可跟踪可监控。
